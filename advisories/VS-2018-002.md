#### CVE ID
CVE-2018-6822

#### CVSS Score

#### Vendor
PureVPN

#### Product
PureVPN for MacOS 6.0.1

#### Vulnerability Details
The PureVPN's **HelperTool** LaunchDaemon implements an unprotected XPC service that can be abused to execute system commands as root.
